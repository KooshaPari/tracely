# tracely Plan

## Overview
Unified observability: tracing, metrics, logging in one Rust library.

## Phases

### Phase 1: Core Wrappers (2 weeks)
- Tracing integration
- Metrics macros
- Logging utilities

### Phase 2: Exporters (2 weeks)
- OTLP exporter
- Prometheus exporter
- Jaeger/Zipkin exporters

### Phase 3: Polish (1 week)
- Configuration builder
- Documentation
- Benchmarks

## Deliverables
- Unified observability API
- OpenTelemetry-compatible spans
- Prometheus metrics export
- Zero-allocation JSON logging
- OTLP, Prometheus, Jaeger, Zipkin exporters
